Hugo github.io 无样式问题

hugo 建站后，生成静态页面并上传github仓库，无法正常显示主题样式

hugo --theme=ananke --baseUrl="http://TomatoAres.github.io/"

将上述命令中的 http 改为 https 即可解决

hugo --theme=ananke --baseUrl="https://TomatoAres.github.io/"
