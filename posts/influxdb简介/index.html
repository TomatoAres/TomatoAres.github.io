Influxdb 简介

概述

  1. 使用 GO 语言开发
  2. 开源
  3. 时序数据库：适合用于处理和分析资源监控数据这种时序相关数据
  4. SQL-like 语法

三大特性：

  1. 时序性（Time Series）：与时间相关的函数的灵活使用（诸如最大、最小、求和等）；
  2. 度量（Metrics）：对实时大量数据进行计算；
  3. 事件（Event）：支持任意的事件数据，换句话说，任意事件的数据我们都可以做操作。

其他特性：

  1. 无结构：schemaless(无结构)，可以是任意数量的列；
  2. 度量方便：min, max, sum, count, mean, median 一系列函数，方便统计；
  3. 支持 http：Native HTTP API, 内置 http 支持，使用 http 读写；
  4. 查询语法简单：Powerful Query Language 类似 sql；
  5. 管理方便：Built-in Explorer 自带管理工具。

概念

数据库相关名词

  1. database：数据库；
  2. measurement：数据库中的表；
  3. points：表里面的一行数据。

特有名词：

Point 由时间戳（time）、数据（field）和标签（tags）组成。

  1. time：每条数据记录的时间，也是数据库自动生成的主索引；
  2. fields：各种记录的值；
  3. tags：各种有索引的属性。
