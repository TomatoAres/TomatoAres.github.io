集群化存储课程总结

2019年12月18日
| Storage
| 阅读

概述

用于存储的网络技术：连接

  1. DAS: Direct Attached Storage
    1. 直接附加存储
    2. 直连主机：通过数据线，光缆，SATA
    3. 优点
      1. 技术简单
      2. 传输效率高
    4. 缺点
      1. 与主机绑定太死，不易于后期扩展
  2. NAS: Network Attached Storage
    1. 网络附加存储
    2. 通过网络与主机相连
    3. 优点
      1. 技术相对简单
      2. 不要求直连主机，局域网即可
    4. 缺点
      1. 传输效率慢
  3. SAN: Storage Area Network
    1. 存储区域网络
    2. 隔离：生产网络——存储网络
    3. 优点
      1. 存储安全性高
      2. 存储效率高
    4. 缺点
      1. 造价高
      2. 技术相对高

实现存储技术分类

  1. 块存储
    1. 常见：
      1. U 盘，磁盘，光驱
      2. ISCSI
    2. 优点
      1. 可独立使用
      2. 通过 Raid，LVM 等简单技术实现高可用
    3. 缺点
      1. 不利于多台设备之间共享
  2. 文件存储、网络存储
    1. 常见
      1. SAMBA
      2. FTP
      3. NFS
    2. 优点
      1. 便宜
      2. 可共享
    3. 缺点
      1. 读写效率低
      2. 传输效率慢
  3. 对象存储、分布式存储、存储桶
    1. 常见
      1. CEPH
      2. MooseFS
      3. RHCS
      4. ClusterFS
    2. 优点
      1. 读写效率高
      2. 可在不同主机间共享
    3. 缺点
      1. 贵
      2. 难：技术难度高

存储技术与存储网络的关系

  1. 块存储+DAS
    1. 最常见 电脑插磁盘
    2. 是所有的基础
  2. 块存储 + NAS
    1. 局域网共享
    2. 提升空间利用率
    3. 便于后期扩展
  3. 块存储 + SAN
    1. 共享
    2. 提升效率
    3. 便于扩展
    4. 更加解耦，更易维护
  4. 文件存储 + NAS
    1. 同 2
  5. 对象存储 + SAN
    1. 利用分布式存储结构——提升读写并发效率
    2. 隔离，更易扩展

块存储——ISCSI

  • 定义：
    • iSCSI（Internet Small Computer System Interface，发音为/ˈаɪskʌzi/），Internet 小型计算机系统接口，又称为 IP-SAN，是一种基于因特网及 SCSI-3 协议下的存储技术
    • SCSI 小型计算机系统接口（SCSI，Small Computer System Interface）是一种用于计算机及其周边设备之间（硬盘、软驱、光驱、打印机、扫描仪等）系统级接口的独立处理器标准

文件存储——NFS

  • 是什么：网络文件系统
  • 功能：网络共享文件
  • 原理：RPC 负载信息的传输
  • 优点
    • 节省空间
    • 共享

安装配置

跳过——暂时不用

分布式存储——MFS

  • 定义：分布式文件系统
    • 具有冗余容错功能
    • 分布式存储——多副本
  • 特性——优点
    • 高可靠：多副本
    • 高扩展：加磁盘，加主机都行
    • 高容错：回收站机制
    • 高数据一致性
  • 缺点
    • 单 master
    • 硬件要求高
    • 复制元数据时间长
  • 应用场景
    • 大规模高并发
    • 不要求性能
