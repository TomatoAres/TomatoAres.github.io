Linux Control Group 简介

2019年9月26日
| os
| 阅读

Linux Control Group(简称 cgroup)是一个用于限制、统计和隔离进程的资源的特性

在虚拟化领域，如 qemu-kvm 和 linux container，cgroup 用常用来限制以下类型的资源：

  1. CPU time：进程占用的 CPU 时间
  2. Memory：进程占用的物理内存
  3. Block IO：进程访问块设备的 bandwidth 或 IOPS
  4. Network IO：进程访问网络的 bandwidth 或 packages 的数量
